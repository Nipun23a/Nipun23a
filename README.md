## Featured Research & Systems
 
### AIM-IPS — Adaptive Intrusion Prevention System
> *Accepted at SciSec 2026 · Springer LNCS · Live at [aimips.tech](https://aimips.tech)*
 
A production-deployed middleware IPS featuring:
- **5-layer ML pipeline** — CNN anomaly gate + LightGBM classifier + cross-pipeline temporal correlator
- **Zero-day detection** via heterogeneous unsupervised ensemble (Autoencoder, VAE, One-Class SVM, Isolation Forest)
- **LLM-powered autonomous agent** — generates adaptive MITRE ATT&CK-mapped WAF rules in real time
- **Graduated response engine** — 0–100 risk scoring with delay / throttle / challenge / block actions under **10ms**
### ShieldSync — WordPress WAF & IPS
> *Open-source · [getshieldsync.com](https://getshieldsync.com) · [github.com/shieldsync/plugin](https://github.com/shieldsync/plugin)*
 
Pre-init WordPress firewall that intercepts every HTTP request before the CMS loads:
- Protects against SQLi, XSS, CSRF, brute force, bot traffic, and file upload attacks
- Pro cloud backend: Laravel + PostgreSQL + Redis + Laravel Queues + AI/ML anomaly detection
- Global crowdsourced IP reputation network
